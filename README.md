# Ad'eux Coaching — Site internet

Site vitrine d'Ad'eux Coaching : accompagnement individuel pour femmes en transition professionnelle. 4 pages statiques (HTML/CSS), 100 % autonomes.

## Pages
- `index.html` — Accueil
- `comment-ca-marche.html` — Comment ça marche
- `offres.html` — Mes offres
- `about.html` — Mon histoire

## Aperçu en local
Ouvre simplement `index.html` dans ton navigateur (double-clic).

## Mise en ligne gratuite avec GitHub Pages
1. Crée un nouveau dépôt sur GitHub (ex. `site-adeux`).
2. Envoie-y tous les fichiers de ce dossier (bouton **Add file › Upload files**, puis glisse-dépose le contenu).
3. Va dans **Settings › Pages**.
4. Sous **Branch**, choisis `main` puis `/ (root)` et clique **Save**.
5. Après une minute, ton site est en ligne à l'adresse :
   `https://<ton-identifiant>.github.io/site-adeux/`

## À finaliser avant la mise en ligne réelle
- **Lien de réservation** : les boutons pointent pour l'instant vers `mailto:contact@adeux-coaching.fr`. Remplacer par le vrai lien Calendly (ou autre) partout où figure ce mailto.
- **Mentions légales & confidentialité** : liens présents mais pages à rédiger.
- **Témoignages** : textes d'exemple à remplacer par de vrais témoignages.
